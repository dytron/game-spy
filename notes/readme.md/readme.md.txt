**Name convention**
Sprites		=> spr_name
Objects		=> obj_name
	- Variables		=> variable_name
	- Methods		=> methodName()
Rooms		=> rm_name
Scripts		=> Name
	- Functions		=> function_name()
	- Classes		=> ClassName()
		- Variables and Methods follow Object name convention
Sounds		=> snd_name
